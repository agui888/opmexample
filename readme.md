opmexample.lua
===

[opm](http://opm.openresty.org/) example package, also the most simple opm package


Install
---

```bash
opm get chunpu/opmexample
```


Usage
---

```lua
local example = require 'opmexample'
print(example.add(1, 2)) -- print 3
```
